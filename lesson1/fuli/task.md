## 阶段二：互动计算与数据可视化（4 个任务）

**目标**：用户可以调整参数，实时看到复利 vs 单利的曲线对比。

---

### 任务 2.1：复利计算器 — 参数控制面板 ✅

- [x] 三个 Range 滑块组件（自定义样式，金色主题）：
  - 本金：100 ~ 1,000,000 元（对数刻度）
  - 年化收益率：1% ~ 30%（步长 0.5%）
  - 时间：1 ~ 50 年（步长 1 年）
- [x] 每个滑块旁显示当前数值（大号字体）
- [x] 实时显示计算结果卡片：最终金额、总收益、收益倍数
- [x] 输入框也支持直接输入数值（与滑块双向绑定）

**验收标准**：拖动任意滑块，三个数值卡片实时更新且计算正确。✅

---

### 任务 2.2：复利 vs 单利 — Chart.js 对比曲线 ✅

- [x] 使用 `react-chartjs-2` 绘制双曲线图：
  - 金色曲线 = 复利收益（下方渐变填充）
  - 灰色虚线 = 单利收益
  - X 轴 = 年份（1 ~ 当前选择年数），Y 轴 = 金额
- [x] 参数改变时曲线平滑过渡动画
- [x] 悬停时显示 tooltip（年份 + 复利金额 + 单利金额 + 差额）
- [x] 图表右上角标注"复利比单利多赚 ¥xxx"

**验收标准**：调整参数后曲线实时更新，悬停可看到数据点详情。✅

---

### 任务 2.3：72 法则标记与数据标注 ✅

- [x] 在曲线上自动计算并标注"翻倍点"（本金翻 2×, 4×, 8×）
- [x] 翻倍点用小圆点 + 标签标注（如"6.1年 → 20万翻倍"）
- [x] 显示当前参数下 72 法则的近似翻倍年限
- [x] 曲线下方展示关键年份的数据表格（第 1、5、10、20、30、50 年）

**验收标准**：曲线上可见翻倍标记点，72 法则计算正确。✅

---

### 任务 2.4：定投计算器（DCA 子面板） ✅

- [x] 切换 Tab：一次性投入 / 每月定投
- [x] DCA 参数：每月投入金额 + 年限 + 收益率
- [x] 定投复利曲线（累计投入 vs 最终市值）
- [x] 对比卡片："一次性投入 ¥X，最终 ¥Y" vs "每月定投 ¥Z，累计投入 ¥W，最终 ¥V"
- [x] 柱状图对比两种方式的最终收益

**验收标准**：切换到定投模式后，输入参数可看到正确的定投复利计算结果。✅
