# 复利学习平台 — 功能规格说明书

## 一、项目定位

打造一个**深入、全面、多角度**的复利概念学习平台，以纯前端网页为载体，结合动态演示、名人名言、外部搜索等模块，让用户从数学原理、历史案例、投资实践、心理学等多个维度透彻理解"复利"。

---

## 二、设计思路

### 2.1 核心理念

- **视觉优先**：用图表和动画让抽象的数学概念变得直观可感
- **多角度穿透**：同一个概念从公式、曲线、名言、故事、对比实验多个切口同时进入
- **渐进式深度**：默认展示最直观的动画，用户滚动/点击即可层层深入

### 2.2 信息架构

```
首页（复利曲线动画 + 一句核心名言）
 ├── 角度一：数学与直觉（互动计算器 + 动态曲线）
 ├── 角度二：历史与故事（时间线 + 名人案例）
 ├── 角度三：名言与智慧（名言墙 + 人物背景）
 ├── 角度四：对比与陷阱（单利 vs 复利、负复利通胀）
 ├── 角度五：生活与哲学（知识复利、习惯复利、关系复利）
 └── 外部资源（搜索入口 + 推荐链接）
```

### 2.3 技术选型建议

| 层次 | 选型 |
|------|------|
| 框架 | 纯 HTML/CSS/JS 或 React + Vite（轻量） |
| 图表 | Chart.js 或 ECharts（动画丰富、交互好） |
| 样式 | Tailwind CSS / 纯 CSS（暗色主题，专业感） |
| 外部搜索 | 调用 DuckDuckGo / Google 搜索 API 或内嵌搜索组件 |
| 部署 | 静态站点，GitHub Pages / Vercel |

## 三、功能模块

### 3.1 首页 — 引力场入口

- **全屏动画**：一个不断生长的复利曲线从零缓缓攀升，曲线尾部陡峭上扬
- **核心名言轮播**：在曲线下方淡入淡出展示爱因斯坦、巴菲特等名言
- **滚动引导**：轻提示"向下滚动，探索复利的秘密"

### 3.2 角度一：数学与直觉（互动计算器）

**核心功能 — 复利互动计算器**：

- 用户可拖动滑块调整三个核心参数：
  - 本金（100 ~ 1,000,000 元）
  - 年化收益率（1% ~ 30%）
  - 时间（1 ~ 50 年）
- **实时反馈曲线**：参数调整时曲线平滑变形
- **终点数值**：大号字体突出显示最终金额
- **双重对比曲线**：
  - 蓝色曲线 = 复利收益
  - 灰色曲线 = 单利收益
  - 两者差距随年份越来越大，视觉冲击明显
- **72法则演示**：在曲线上标注"此处翻倍"节点
- **关键数据标注**：悬停/点击曲线上任意点，显示该年的本金+收益明细

**子面板 — 定投计算器**：

- 每月定投金额 + 年限 + 收益率 → 展示定投复利效果
- 对比"一次性投入" vs "每月定投"的终值差异

### 3.3 角度二：历史与故事

**时间线组件**：

- 横轴 = 时间（近现代 → 当代）
- 关键节点包含：
  - 巴菲特年化 20% 持续 60 年的财富增长曲线
  - 曼哈顿岛 24 美元卖出的复利反推 → 印第安人亏了多少
  - 棋盘麦粒问题（2^64 的爆炸增长）
  - 美国股市百年增长（S&P 500 回测数据）
  - 日本失去的三十年（零利率对复利的反向案例）

**每个节点点击展开**：详细故事 + 复利计算演示 + 延伸思考

### 3.4 角度三：名言与智慧

**名言卡片墙**：

| 人物 | 名言 | 背景解读 |
|------|------|---------|
| 爱因斯坦 | "复利是世界第八大奇迹" | 物理学家对数学规律的惊叹 |
| 巴菲特 | "人生就像滚雪球，重要的是找到湿的雪和很长的坡" | 好公司 + 长期持有 |
| 芒格 | "理解复利的威力和获得它的难度，是理解许多事情的核心" | 逆向思维 |
| 本杰明·富兰克林 | "钱能生钱，生出来的钱又能生更多" | 200年前的智慧 |
| 约翰·博格 | "复利是投资者最好的朋友" | 指数基金之父 |

- 每张卡片 = 头像 + 名言 + 人物简介 + **相关复利数值演示**
- 每个名言的数学换算：比如巴菲特的雪球 → 可视化"本金 × (1+收益率)^年数"的雪球滚大动画

### 3.5 角度四：对比与陷阱

**对比实验面板**：

- **单利 vs 复利**：同本金、同利率、同时长，差距随时间指数级拉大
- **早开始 vs 晚开始**：A 从 25 岁开始投资 10 年，B 从 35 岁开始投资 30 年，谁最终更多？
- **费率陷阱**：1% 管理费在 30 年内吃掉多少收益？（差值可高达 30%+）
- **通胀侵蚀**：名义收益率 vs 实际收益率的对比曲线
- **负复利**：信用卡 18% 年利率的债务雪球

### 3.6 角度五：生活与哲学

**复利思维的跨界应用**：

- **知识复利**：每天学一点，知识网络如何指数增长（概念连线动画）
- **习惯复利**：每天进步 1%，一年后是 37.8 倍；每天退步 1%，一年后只剩 0.03
- **关系复利**：信任如何通过一次次微小承诺积累

**1% 法则动画**：

```
1.01^365 = 37.8
0.99^365 = 0.03
```

- 用粒子动画展示两个数列的演化路径

### 3.7 外部搜索 & 资源

- **内嵌搜索框**：输入关键词，调用 DuckDuckGo 搜索"复利 X"
- **推荐资源列表**：
  - 《穷查理宝典》
  - 《巴菲特致股东的信》
  - 《复利的力量》（The Power of Compound Interest）
  - John Bogle 的演讲视频链接
- **一键分享**：生成当前计算结果的截图或链接

## 四、非功能需求

### 4.1 性能
- 首屏加载 < 2s
- 图表动画 60fps
- 移动端响应式适配

### 4.2 可访问性
- 支持键盘导航
- 图表数据提供文字描述
- 颜色对比度符合 WCAG AA

### 4.3 国际化
- 默认中文，预留 i18n 扩展能力
